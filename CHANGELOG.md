# Changelog

All notable changes to PyEnTT will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.4] - 2025-03-05

### Added
- Multi-version Python support (3.10, 3.11, 3.12, 3.13, 3.14) using stable ABI
- GitHub Actions CI/CD workflows
- LICENSE (MIT)
- README.md with comprehensive documentation
- CONTRIBUTING.md with contribution guidelines
- CHANGELOG.md for version history

### Changed
- Updated Python version requirement from 3.14 to 3.10
- Changed build type from Debug to Release for production builds
- Updated package description
- Removed personal PyPI repository configuration

### Fixed
- Multi-platform build support (Linux, macOS, Windows)

## [0.1.3] - 2024-XX-XX

### Added
- Initial EnTT Python bindings
- Entity management
- Component system
- Registry operations
- Observer pattern implementation
- Reactive storage support

### Changed
- Initial release

## [Unreleased]

### Planned
- Enhanced documentation
- More examples and tutorials
- Performance optimizations
- Additional ECS features
